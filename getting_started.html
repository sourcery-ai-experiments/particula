Installing particula¶

If you have no idea what Python is, or if you have no idea what a Jupyter notebook is, or if you have no clue what is going at all, then this is probably a good a place to start!

Python & particula¶

Python is a popular programming language that is remarkably easy to learn and use. It is also a very popular language for scientific computing with a lot of applications and libraries, comprising a large ecosystem of packages. We hope particula will be a useful tool in this large ecosystem for anyone who wants to learn more about or research the science of aerosol particles.

Jupyter notebooks¶

Jupyter notebooks are a popular way to organize and share Python code and output with others. To learn more about Jupyter notebooks, see here. The idea is that you can write code in a a code cell, and then run said cell to execute the code through a kernel (a computer program that runs Python code). Besides code cells, you can also write markdown (text) cells that can be displayed like the text you are reading. You can add rendered mathematical symbols, \(\sqrt{x} = y\), or even images.

Install particula¶

Prepackaged¶

The particula package is as a Python package through the pip package manager. Simply run the following command in a terminal to install the package: pip install -U particula. In a notebook like this, one would run the following command: %pip install -U particula, where % is the Jupyter magic command to run a cell as if it were a command in terminal. The flag -U means “update the package if it is already installed”.

We also have particula available as a conda package from the conda-forge channel. If you have a flavor of conda installed (e.g. anaconda, miniconda, miniforge, or mambaforge), you can install the package with the following command: conda install -c conda-forge particula. If you do not have conda installed, please see the following to get started: https://conda.io/docs/user-guide/install/. We recommend mambaforge from the miniforge repository, available here.

From source¶

You can obtain particula from the source repository. To install the package, you will need to have a Python 3.7 or newer version installed. You then should install the basic dependency packages with the following commands: pip install -r requirements.txt or conda install --file requirements.txt for development purposes. If you intend to only use the package, for now, you only need numpy and pint. We will be providing more information soon.
